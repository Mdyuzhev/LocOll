# Команда /init — инициализация сессии

Сразу написать: **🚀 Начинаю работу**

Выполни последовательно, после каждого шага писать статус:

**Шаг 1:** `start_session` с `project_path` = текущая рабочая директория
→ написать: **✅ Шаг 1 готов — сессия открыта**

**Шаг 2:** Проверь доступность homelab-mcp (сначала LAN, потом Tailscale):
`curl -s --max-time 2 http://192.168.1.74:8765/health`
- Если ответ содержит `"status":"ok"` — написать "🟢 homelab-mcp доступен (LAN)"
- Если LAN недоступен — попробовать Tailscale: `curl -s --max-time 2 http://100.81.243.12:8765/health`
  - Если Tailscale ответил — написать "🟢 homelab-mcp доступен (Tailscale)"
  - Если оба недоступны — написать "🔴 homelab-mcp недоступен (проверь сервер)"
→ написать: **✅ Шаг 2 готов — сервер проверен**

**Шаг 3:** `get_context` — получить полный контекст включая чекпоинты активной сессии
→ написать: **✅ Шаг 3 готов — контекст получен**
