Производственный мониторинг — разовый срез состояния всех проектов.

Выполни:

1. Вызови `mcp__homelab__get_production_status` (since_hours=24)
2. Проанализируй результат:
   - verdict: ALL_GREEN / DEGRADED / CRITICAL
   - Для каждого проекта: сервисы up/down, события, git-статус
3. Выведи краткий отчёт в формате:

```
## Production Status: {verdict}

| Проект | Сервисы | Events | Git | Проблемы |
|--------|---------|--------|-----|----------|
| ...    | N/M up  | K      | abc1234 (clean/dirty) | down: ... |

Время проверки: X.Xs
```

4. Если verdict != ALL_GREEN — предложи конкретные действия для восстановления.
